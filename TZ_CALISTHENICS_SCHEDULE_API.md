# ТЗ: API расписания для Telegram-бота

## Контекст
Telegram-бот I Do Calisthenics использует команду `/reschedule` — пользователь хочет перенести/посмотреть расписание. Сейчас бот отправляет данные в Make.com. Нужно заменить это на получение расписания с сайта calisthenics.ru.

**Цель:** добавить на calisthenics.ru API-эндпоинт, который отдаёт расписание занятий по студии. Бот будет вызывать этот API и показывать расписание пользователю в Telegram.

---

## 1. Endpoint

**Метод:** `GET`  
**URL:** `https://calisthenics.ru/api/schedule`  
**Параметры запроса (query):**
- `studio` (обязательный) — идентификатор студии (см. список ниже)

**Пример:**
```
GET https://calisthenics.ru/api/schedule?studio=msk_youcan
```

---

## 2. Идентификаторы студий

Бот передаёт один из следующих `studio` в зависимости от тега клиента в Airtable:

| studio | Описание (для справки) |
|--------|------------------------|
| `msk_youcan` | м. 1905г., You Can, Москва |
| `msk_elfit` | м. Октябрьская, El Fit, Москва |
| `spb_hkc` | м. Выборгская, Hells Kitchen, СПб |
| `spb_spirit` | м. Московские Ворота, Spirit, СПб |

---

## 3. Формат ответа

**Content-Type:** `application/json`

**Успешный ответ (200 OK):**
```json
{
  "ok": true,
  "studio": "msk_youcan",
  "schedule": "Расписание занятий:\nСб, 07 марта 12:00\nвт, 10 марта 18:40\nвт, 10 марта 20:00\nчт, 12 марта 18:40\nчт, 12 марта 20:00"
}
```

- `schedule` — строка с расписанием. Бот отправит её пользователю как есть. Поддерживаются `\n` для переносов строк.

**Ошибка (неизвестная студия, 404 и т.п.):**
```json
{
  "ok": false,
  "error": "Студия не найдена"
}
```

---

## 4. Требования

1. **CORS:** не обязательно для бота (он делает запросы с сервера Node.js), но желательно разрешить `*` для возможного использования с фронта.
2. **Таймаут:** ответ в течение 2–3 секунд.
3. **Источник данных:** расписание может храниться в JSON/статика на сайте или в CMS — на усмотрение разработчика. Главное, чтобы данные можно было обновлять при изменении расписания.
4. **Язык:** текст в `schedule` — русский.
